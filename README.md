# 桨领航团AI达人创造营第二期-作业4-新鲜红肉识别
> 桨领航团AI达人创造营第二期-作业4-新鲜红肉识别


## 项目描述
> 肉如果不新鲜会影响身体健康，某些厨房新手没有经验不容易分辨肉类是否新鲜，通过计算机视觉方法对肉类进行检测，判断肉是否新鲜。

## 数据介绍
[新鲜和过期红肉数据集](https://aistudio.baidu.com/aistudio/datasetdetail/84831/0)   
数据集介绍：像素1280 x 720. 共计 1896 张图片 每类, 948张  
分为：  
Fresh 新鲜肉  
Spoiled 不新鲜肉
